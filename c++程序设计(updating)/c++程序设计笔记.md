# 1. C++基本概念

**高内聚低耦合**

## 对象：

使用类生成的实例

## 类：

用户定义的一个数据类型

## 抽象：

对象只暴露必要的外部信息，将复杂的内部细节和背景信息隐藏在对象内部

## 封装：

将相关联的数据和函数绑定在同一实体中，保证数据安全

## 继承：

对类的功能进行拓展，有效实现代码复用，使代码结构更清晰移动
是重要多态性的实现方式之一

## 多态：

在类等级的不同层次中可以共享一个行为的名字，不同层级每个类按自己的需求来实现
